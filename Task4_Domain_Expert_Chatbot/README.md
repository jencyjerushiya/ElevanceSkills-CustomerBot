# Domain Expert Chatbot

## Project Overview

The Domain Expert Chatbot is an AI-powered application that provides answers to questions related to Artificial Intelligence and Computer Science concepts. It uses semantic similarity techniques to retrieve the most relevant information from a predefined knowledge base and presents the results through an interactive Streamlit interface.

## Features

* Answers domain-specific questions related to AI and Computer Science.
* Uses semantic search to retrieve relevant information.
* Covers topics such as:

  * Artificial Intelligence
  * Machine Learning
  * Deep Learning
  * Natural Language Processing
  * Computer Vision
  * Large Language Models
  * Reinforcement Learning
* Simple and interactive Streamlit-based user interface.

## Technologies Used

* Python
* Streamlit
* Sentence Transformers
* Semantic Search

## How to Run

1. Install the required libraries.
2. Open a terminal in the project folder.
3. Run:

```bash
streamlit run app.py
```

4. Open the local URL displayed in your browser.
5. Ask questions related to AI and Computer Science concepts.

## Example Questions

* What is Artificial Intelligence?
* Explain Machine Learning.
* What is Deep Learning?
* What are Large Language Models?
* Tell me about Natural Language Processing.
